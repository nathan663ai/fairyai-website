Story Continuation: Build Ongoing Adventures

Stories don't have to end. Continue them night after night to build an epic series your child will love.

How Story Continuation Works

  1. Open Any Saved Story Go to your story library and select a story
  2. Tap "Continue This Story" Button appears at the bottom of each story
  3. AI Picks Up Where You Left Off System automatically includes previous story summary, maintains same tone and style, and creates Chapter 2, 3, 4, etc.
  4. Continue Indefinitely Each continuation costs 1 Fairy Dust—build sagas that last weeks or months

Text-Based Characters

Best for: Creating fantasy characters from imagination.

Character Wizard (Step-by-Step):

  1. Name & Description Enter character name and brief description
  2. Choose Appearance Hair color, eye color, skin tone, species/type
  3. Select Personality Up to 5 traits (brave, kind, curious, etc.)
  4. Generate 6 Art Styles Creates: Cartoon, Realistic, Watercolor, Pixar-style, Anime, Storybook
  5. Choose Your Favorite Select which art style to use (can change later)

Quick Character (Fast):

  1. Name & Description Character name + 50-500 character description

  2. Generate 6 Art Styles Same 6 styles created automatically
  3. Choose Your Favorite Select preferred art style
